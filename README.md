# Nuju.app — AI Journal That Understands Your Life

The 30-second AI journal companion. Track moods, discover patterns, talk to Ju.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Routes

- `/` — Landing page
- `/app` — Main journal app

## Tech Stack

- Next.js 14 + React 18
- Supabase (auth + database)
- Tailwind CSS
- Gemini AI (coming soon)

## Deploy

Push to GitHub → Import in Vercel → Add env vars → Done.
